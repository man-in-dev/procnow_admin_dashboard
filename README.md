# Admin Dashboard - ProcureFlow

A Next.js admin dashboard for ProcureFlow.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Dashboard with enquiries statistics
- Sidebar navigation with Enquiries section
- Responsive design
- Modern UI matching ProcureFlow design
